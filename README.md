# LitCit
Simulation and Visualisation of Laser-Induced Temperature Calculation In Tissue.

# Requirements
* Python3
* venv

# How to run this?
In UNIX system:
```
git clone https://github.com/KNMPPWR/LitCit.git
cd LitCit
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
python3 app.py
```

In Windows:
```
git clone https://github.com/KNMPPWR/LitCit.git
cd LitCit
python3 -m venv env
env\Scripts\activate
pip install -r requirements.txt
python3 app.py
```
